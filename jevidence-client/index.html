Error!

Probably your browser does not allow to load tests execution. There are two ways to solve the problem:

1. Run this report from a local server

2. Change security policy for local files in your browser



Chrome

a) Close all chrome instances

b) Linux users: Start chrome browser with params: --allow-file-access-from-files Eg. chrome --allow-file-access-from-files

b) Windows users: right-click on chrome shourtcut -> Properties -> target. Add the following param: --allow-file-access-from-files

c) Mac OS users: Execute: open /Applications/Google\ Chrome.app --args --allow-file-access-from-files

Safari

a) Open Safari

b) Go to Advanced -> Enable "Show develop menu in menu bar"

c) Go to Develop menu

d) Select: Disable local file restrictions

Firefox

a) Type "about:config" in your url

b) Find "security.fileuri.strict_origin_policy" parameter

c) Set it to false
